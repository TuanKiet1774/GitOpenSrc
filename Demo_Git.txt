Các lệnh Git cơ bản
1. git init -> Khởi tạo 1 Git Repo ở trong máy của mình 
Initialized empty Git repository in... => Tạo Git Repo thành công

2. git status -> Kiểm tra có bất kỳ sự thay đổi nào không
No commits yet => Chưa có bất kỳ sự thay đổi nào (file màu đỏ)

3. git add . -> Cập nhật toàn bộ thay đổi lên Staging Area (khu vực chờ)
Sử dụng git status để kiểm tra (file màu xanh)

4. git commit -m "Nội dung, ghi chú Comit" -> Lưu từ khu vực chờ vào local repository

5. git log -> Để kiểm tra commits 

6. git remote add origin https://github.com/TuanKiet1774/GitOpenSrc.git -> Gắn repo local với repo GitHub từ xa, để sau đó bạn có thể git push (đẩy code lên) hoặc git pull (kéo code về).

7. git branch -M main -> Tạo nhánh main trong Repo trên Github

8. git push -> Sau khi chỉnh sửa thêm, git add . và git commit để thêm thay đổi, ta sẽ sử dụng git push để đẩy code thay đổi lên Github

9. git pull -> Đồng bộ code từ Github về máy 

Nếu xảy ra trường hợp bị xung đột, tức là trên Github có lệnh mà Local không có, thì phải git pull để đồng bộ code về Local rồi mới push lên Github

10. git clone https://github.com/thanhbinh24-peace/gitCourse.git -> Lấy code từ một Repo nào đó về Local
cd gitCourse -> Chuyển đường dẫn sang thư mục gitCourse
dir -> Để xem các file con bên trong

11. git branch -> Xem nhánh hiện tại

12. git branch Branch1 -> Tạo một nhánh tên là Branch1

13. git checkout Branch1 -> Chuyển sang làm việc trên nhánh Branch1
Sau khi tạo nhánh mới, để đẩy code lên nhánh này thì cần phải chỉ rõ đẩy nhánh này lên remote với tên gì (Lần đầu)
=> git push --set-upstream origin Branch1

git checkout main -> Quay lại làm việc trên nhánh main

14. git merge Branch1 -> Trộn nhánh main vào vị trí nhánh hiện tại Branch1
Đảm bảo đang ở nhánh cần trộn ví dụ nhánh main => git checkout main => git merge Branch1
Sau khi merge ==> git add . và git push để đẩy code lên Github. 
Vì merge đã tự động commit khi trộn
